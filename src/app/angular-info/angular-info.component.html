Angular Versions

Angular JS
Advantages:
  • Efficient framework that can create Rich Internet Applications (RIA)
  • Write client side applications using JavaScript in a clean Model View Controller (MVC) way
  • Two-way data binding
  • DOM manipulation
  • Faster application prototyping
  • Responsive web
Disadvantages:
  • JavaScript support is mandatory
  • Inexperience with MVC
  • Use of scopes
  • Learning difficulties
Angular 2.0
Advantages:
  • Mobile development
  • Better performance
  • Supports routing
  • Cross platform
  • Syntax is comparatively easier than AngularJS
Angular 4.0
Advantages:
  • HttpClient, a more powerful library for making HTTP Requests
  • Router life cycle events for Guards and Resolvers
  • Conditionally disable animations
Angular 5.0
Advantages:
  • Support for progressive web apps
  • Build optimizer
  • Improvements related to Material Design
Angular 6.0
Advantages:
  • Focus more on the toolchain and on making it easier to move quickly with Angular in the future, like: ng update, ng add, Angular Elements, Angular Material + CDK Components, Angular Material Starter Components, CLI Workspaces, Library Support, Tree Shakable Providers, Animations Performance Improvements, and RxJS v6
Angular 7.0
Advantages:
  • Updates regarding Application Performance, Angular Material & CDK, Virtual Scrolling, Improved Accessibility of Selects
  • Supports Content Projection using web standard for custom elements
  • Dependency updates regarding Typescript 3.1, RxJS 6.3, Node 10
Angular 8.0
Advantages:
  • Generated code that is easier to read and debug at runtime
  • Faster re-build time
  • Improved payload size
  • Improved template type checking
  • Backwards compatibility
Angular 9.0
Advantages:
  • Smaller bundle sizes
  • Faster testing
  • Better debugging
  • Improved CSS class and style binding
  • Improved type checking
  • Improved build errors
  • Improved build times, enabling AOT on by default
Angular 10.0
Advantages:
  • New Date Range Picker (Material UI library)
  • Warnings about CommonJS imports
  • Optional Stricter Settings
  • Keeping Up to Date with the Ecosystem
  • New Default Browser Configuration
  • Deprecations and Removals
